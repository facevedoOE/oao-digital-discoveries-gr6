Q1. A simulation is BEST described as:
───────────────────────────────────────────────────────────────
  A) A wild guess
     ❌ Simulations are structured runs of a model.
  B) What happens when a digital model is run                  ✓ CORRECT
     ✅ A simulation is the running of a model. (SC.6.PE.3.1)
  C) A static picture
     ❌ Static images don't change. Simulations show change.
  D) A list of facts
     ❌ Data alone isn't a simulation.

───────────────────────────────────────────────────────────────
Q2. Which is an example of a CONDITION?
───────────────────────────────────────────────────────────────
  A) score = 0
     ❌ Variable assignment, not a condition.
  B) IF score > 100 THEN level up                              ✓ CORRECT
     ✅ Textbook IF/THEN condition. (SC.6.PE.3.1)
  C) player_name = "Alex"
     ❌ Assignment.
  D) lives = 3
     ❌ Assignment.

     ✅ Information you feel through vibration. (SC.6.ET.1.1)

───────────────────────────────────────────────────────────────
  A) Ship anyway                     ❌ Hoping isn't engineering.

     ✅ Test → Improve loop. (SC.6.PE.3.3)
  D) Hide the bug                    ❌ Fails users.
