File Types & Digital Organization

Behind every file you open is a tiny three-letter clue — the extension — that tells your computer what kind of thing it is. This week: cracking the code, picking the right format, and keeping your digital life tidy.

Learning Objectives

By the end of this week, you will be able to:

1
Identify common file types across the five major families: documents, images, audio, video, and 3D.
2
Read a file extension and explain what kind of file it points to and what app opens it.
3
Choose the right file format for a project, balancing quality, size, and audience needs.
4
Organize files with folders and clear naming so future-you (and your collaborators) can find anything in seconds.
01
Section 1 · What Is a File?

A file is a labeled box of data

Everything on your computer — from your homework essay to your favorite song — is stored as a file. A file is just a labeled package of digital information. The label tells the computer two things: what to call it, and what kind of thing it is.

🔍 Anatomy of a Filename
my-essay.docx
↑ Name
↑ Extension

The name is for you — describe what's inside. The extension (the dot + 3–4 letters at the end) is for the computer — it says "open me with this kind of app." Change the extension and the file may not open at all.

📘 Vocabulary

File — a single named package of digital information stored on a device or in the cloud.

File extension — the dot + letters at the end of a filename (.docx, .jpg, .mp3) that tell the computer what type of file it is.

File format — the rules a file follows for how its data is arranged. Each extension matches one format.

02
Section 2 · The Five File Families

Meet the file families you'll work with

Most files fall into one of five big families. Each family stores a different kind of content, and each family has a few common formats. You don't need to memorize every extension — just know the family it belongs to.

📄Family 1 · Documents (text)
Essays, reports, letters, slides — anything mostly made of words.
.docx .pdf .txt .pptx .gdoc
🖼️Family 2 · Images
Photos, drawings, icons, screenshots, infographics.
.jpg .png .gif .svg .webp
🔊Family 3 · Audio
Music, podcasts, voice memos, sound effects.
.mp3 .wav .m4a .aac
🎬Family 4 · Video
Movies, recorded lessons, animations, screen recordings.
.mp4 .mov .webm .avi
🧊Family 5 · 3D Models & CAD
3D objects, game characters, 3D printer plans, engineering designs.
.obj .stl .fbx .glb
💡 Did You Know?

There are over 5,000 file extensions in active use today — but you'll only meet about 20 in everyday school and personal life. The rest are specialized formats for engineers, scientists, designers, and game developers.

03
Section 3 · Choosing the Right Format

Same content, different formats — why?

Inside a single family, different formats trade off three things: quality, file size, and compatibility (which apps can open it). Smart choices depend on what you need most.

If you need…
Choose…
Why
A document anyone can open and that won't change layout
.pdf
Locks in fonts and layout; opens on any device.
A photo for a website (small file size)
.jpg or .webp
Compressed for fast loading; great for photographs.
A logo with a transparent background
.png or .svg
Both support transparency. SVG also scales without blurring.
A song to share with friends
.mp3
Small files, plays everywhere — the universal music format.
A video to upload to YouTube or Canvas
.mp4
Most-supported video format on the planet.
💭 Think About It

A photo saved as a giant .png might look beautiful — but if it's 30 MB, your teacher's email won't accept it. Choosing a format is a small act of prudence: thinking about your audience, the situation, and the right tool for the moment.

Section 4 · Digital Organization

Future-you will thank you

📝 The 4 Rules of Good File Names
1
Be specific.
Not "essay.docx" — try civil-war-essay-final.docx.
2
No spaces.
Use hyphens or underscores: my-poster.png beats my poster.png.
3
Add a date or version.
notes-2026-04-27.docx or poster-v2.png.
4
Keep the extension.
Don't rename song.mp3 to song — your computer needs the extension to know how to open it.
✅ Check for Understanding

Open your own downloads folder right now (mentally is fine!). How many files have unhelpful names like "Untitled" or "Document(7)"? What would you rename three of them to be more specific?

05
Section 5 · Why Medium Matters

"The medium is the message"

In the 1960s, a thinker named Marshall McLuhan coined a famous line: "the medium is the message." He meant: how you deliver something changes how it lands. A handwritten letter feels different than the same words in a text. A printed photo feels different than the same image on a screen.

Choosing a file format is a small version of that decision. The same idea sent as a glossy PDF lands differently than the same idea sent as a typo-ridden text. The format shapes how serious, how casual, how trustworthy your message feels.

🌎 Real-World Connection

When you apply for a job one day, you'll be asked for a resume in PDF format — not Word, not a screenshot, not a Google link. Why? PDFs lock the layout, work on every device, and signal professionalism. The format is part of the message.

Doctors send X-rays in .dcm. Engineers send blueprints in .dwg. Every profession has format conventions — and learning to match the format to the situation is part of digital literacy.

📚 Key Vocabulary — Week 5

File
A single named package of digital information stored on a device or in the cloud.
File extension
The dot + letters at the end of a filename that tell the computer the file's type.
File format
The rules a file follows for how its data is organized internally.
PDF
A document format that locks layout and fonts, opens on any device.
Compression
Shrinking a file's size to save space or speed up sharing.
Folder hierarchy
An organized tree of folders inside folders that groups related files.
Naming convention
A consistent rule you follow when naming files — specific, no spaces, dated.
Cloud storage
Storing files on remote servers (Google Drive, OneDrive) instead of just one device.
🪞 Weekly Reflection
Audit and tidy your digital life.

Take 15 minutes this week to:

  • Open your downloads or documents folder. Count files with unhelpful names ("Untitled," "Document(3)").
  • Pick 5 files and rename them following the 4 rules of good file names.
  • Create at least one new folder with a clear name and move related files into it.
  • In 3–4 sentences, describe what you changed and how the new system feels different from the old chaos.
