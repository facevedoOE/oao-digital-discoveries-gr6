───────────────────────────────────────────────────────────────
Q1. (Week 1) Which is the BEST example of SYNCHRONOUS digital
    communication?
───────────────────────────────────────────────────────────────
  A) An email sent overnight
     ❌ Email is asynchronous — sender and receiver aren't online
        together.
  B) A discussion board post replied to next morning
     ❌ Discussion posts are asynchronous — read whenever the
        reader gets to them.
  C) A live video class with your teacher and 20 classmates    ✓ CORRECT
     ✅ Live video is synchronous — everyone is present and
        communicating in real time. (SC.6.CC.1.1)
  D) A voicemail
     ❌ Voicemails are asynchronous — listened to later.

───────────────────────────────────────────────────────────────
Q2. (Week 3) Aristotle's three classical rhetorical purposes
    are:
───────────────────────────────────────────────────────────────
  A) Inform, confuse, frustrate
     ❌ Confusion and frustration aren't real purposes — and
        good designers avoid both!
  B) Inform, persuade, entertain                                ✓ CORRECT
     ✅ Aristotle's framework, still used today: teach the
        audience (inform), change what they think or do
        (persuade), or delight them (entertain). (SC.6.CC.2.1)
  C) Type, click, scroll
     ❌ Those are user actions, not communication purposes.
  D) Sketch, code, present
     ❌ Those are creative steps, not rhetorical purposes.

───────────────────────────────────────────────────────────────
Q3. (Week 4) Aesthetic unity in a multimedia project means:
───────────────────────────────────────────────────────────────
  A) Every page uses different colors and fonts to keep things
     interesting
     ❌ The opposite! Variety on every page kills unity.
  B) Including as many colors and fonts as possible
     ❌ More isn't better. Aesthetic unity comes from RESTRAINT.
  C) Every page feels like part of the same project — same     ✓ CORRECT
     colors, same fonts, same layout style
     ✅ Right! Aesthetic unity is the visual signal that one
        designer made one project. (SC.6.CC.2.1)
  D) Using the most expensive design software
     ❌ Tools don't create unity. Discipline does.

───────────────────────────────────────────────────────────────
Q4. (Week 5) A file extension (like .docx or .mp3) tells the
    computer:
───────────────────────────────────────────────────────────────
  A) The author's name
     ❌ Authors aren't part of the extension.
  B) Where the file is stored
     ❌ Storage location is in the file path, not the extension.
  C) What kind of file it is and which app should open it      ✓ CORRECT
     ✅ Right! Extensions match files to apps. Without the
        extension, the computer often can't tell what to do.
        (SC.6.CO.1.1)
  D) The size of the file
     ❌ File size is shown separately. Extensions describe
        TYPE, not size.

───────────────────────────────────────────────────────────────
Q5. (Week 6) Your teammate accidentally deleted three
    paragraphs of your shared Google Doc. Which feature can
    rescue the deleted text?
───────────────────────────────────────────────────────────────
  A) Real-time co-editing
     ❌ Co-editing is for typing simultaneously — doesn't
        rewind history.
  B) Comments
     ❌ Comments add notes; they don't restore deleted text.
  C) Version history                                           ✓ CORRECT
     ✅ Right! Version history is an automatic record of past
        versions. Find a version from before the accident and
        restore. The ultimate undo. (SC.6.CO.1.2)
  D) Delete the document and start over
     ❌ That makes it worse — you'd lose all the GOOD work too.
